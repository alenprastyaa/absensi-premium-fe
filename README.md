# Administrasi Guru Premium

Frontend React ini terhubung ke backend Node.js di folder `../BE` lewat `VITE_API_BASE_URL`.

## Run Lokal

**Prasyarat:** Node.js dan MySQL

1. Install dependency frontend:
   `npm install`
2. Install dependency backend:
   `cd ../BE && npm install`
3. Salin `FE/.env.example` menjadi `.env` dan `BE/.env.example` menjadi `.env`
4. Jalankan frontend:
   `npm run dev`
5. Jalankan backend dari folder `BE` di terminal lain:
   `npm run dev`

Frontend berjalan di port `3227`. Backend berjalan di port `3401`.
